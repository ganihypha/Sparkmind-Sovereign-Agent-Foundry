buka kalender, buat konten pakai stack B10-04, posting, lalu jalankan cold DM (B10-03) paralel.

**Target 30 hari (realistis, fase-0):**

| Metrik | Target D30 |

> Konten membangun **kredibilitas**; cold DM membawa **closing**. Keduanya jalan bersama.

## 4. Funnel VSL faceless (penonton → checkout)

```
KONTEN (Reels/Feed)  →  PROFIL (bio: 1 kalimat + 1 link)  →  LANDING/DEMO  →  CHECKOUT
      ▲                                                                          │
      └────────────────  Cold DM (B10-03) mengalirkan ke titik yang sama  ◀──────┘
```

**4.1 Bio profil (IG/FB) — 1 kalimat + 1 link:**
```
Outcome Foundry ⚙ Kami bangunkan hasil (kasir, booking, toko online) untuk usaha lokal.
👇 Pilih outcome kamu
outcome-foundry.biz.id
```

**4.2 VSL faceless (60–90 detik, struktur):**
```
0–5s   Hook      : "Booking barbershop masih lewat DM? Ini masalahnya."
5–25s  Problem   : jadwal bentrok, no-show, kas bocor (b-roll AI + teks)
25–55s Solusi+Demo: screen-record app kasir+booking jalan (proof)
55–75s Bukti+Offer: "contoh yang kami bangun" + apa yang didapat
75–90s CTA       : "Link di bio — pilih outcome kamu. Atau DM 'BOOKING'."
```
> Produksi: script (Gemini) → b-roll (AI Studio/Dreamina) → screen-record demo →
> rangkai + TTS + caption (CapCut). Faceless 100%. Lihat **B10-04 §4**.

**4.3 Link di bio:** arahkan ke `outcome-foundry.biz.id/foundry` (katalog) atau halaman demo
flagship. **HITL owner** untuk teks/harga customer-facing.
