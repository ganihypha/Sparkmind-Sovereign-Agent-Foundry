## 1. Prinsip pemilihan tool (checklist)

Sebelum pakai satu tool, cek 5 hal:

1. **No credit card?** Bisa daftar & pakai tanpa isi kartu? (Prioritas: ya.)
2. **Watermark?** Output free-tier ada watermark? (Untuk konten jualan → hindari, atau pilih yang bisa dilepas.)
3. **Hak komersial?** Boleh dipakai untuk promosi/jualan? (Baca ToS — banyak free-tier hanya "personal use".)
4. **Batas harian/bulanan?** Cukup untuk ritme posting (lihat B10-05)?
5. **Ekspor rasio 9:16 & 1:1?** Wajib untuk Reels/Story/Feed.

> Jika sebuah tool gagal di poin 1/2/3 untuk konten jualan → jadikan **draft-only** (bikin ide),
> lalu finalisasi di tool aman (Canva/CapCut).

## 3. Stack rekomendasi (minimal, aman, gratis)

> Pilih **satu** per fungsi supaya tidak kewalahan:

```
Ide/script   →  Gemini (Google AI Studio)
Gambar       →  Google AI Studio  +  Ideogram (khusus gambar berteks)
Video b-roll →  Dreamina  (aksen)  |  Higgsfield (verifikasi ToS dulu)
Desain/poster→  Canva Free
Edit video   →  CapCut  (auto-caption + ekspor bersih)
Voice-over   →  ElevenLabs Free  /  CapCut TTS
Musik        →  CapCut library / YouTube Audio Library
```

**Semua di atas = no-credit-card & ekspor tanpa watermark** (kecuali Pika/Higgsfield free-tier —
pakai hanya untuk aksen atau setelah verifikasi).

## 4. Workflow konten faceless (dari ide → posting, ±30 menit)

```
1. Gemini      → minta 1 script hook+isi+CTA (niche barbershop, Bahasa Indonesia)
2. AI Studio   → generate 3–5 gambar (mockup app, before/after, hero)
3. Dreamina    → 2–3 klip b-roll 3–5 detik dari gambar (opsional)
4. CapCut      → rangkai gambar+klip, tambah auto-caption, TTS voice-over, musik
5. Canva       → bikin thumbnail/cover + carousel edukasi (jika format feed)
6. Ekspor      → 9:16 (Reels/Story) & 1:1 (Feed), tanpa watermark
7. Posting     → caption dari Gemini + CTA "Link di bio — pilih outcome kamu"
```

> **Faceless 100%:** narasi = TTS + teks di layar + b-roll AI + screen-recording demo produk.
> Wajah tidak pernah dibutuhkan.

## 6. Batasan & Truth-Lock konten

- **Verifikasi limit/ToS tiap tool** sebelum dipakai untuk konten jualan (limit berubah).
- **Jangan** klaim visual AI sebagai "foto klien nyata". Demo = "contoh yang kami bangun".
- **Watermark:** hindari tool yang memaksa watermark di free-tier untuk konten jualan (pilih Canva/CapCut untuk finalisasi).
- **Hak komersial:** pastikan output boleh dipakai komersial; bila ragu, jadikan tool itu draft-only.
- **Kredit-aware:** habiskan kuota gratis harian secara efisien; naik berbayar hanya setelah ada revenue.

> Lanjut: kalender konten 30 hari siap-tempel + funnel VSL faceless → **B10-05**.
