## 4. Unit economics (model)

| Variabel | Asumsi konservatif |
|---|---|
| Biaya infra | ~Rp 0 (Cloudflare free-tier) |
| Biaya transaksi | MDR Duitku (QRIS ~0.7%, VA flat) |
| Margin produk digital | ~95%+ (COGS mendekati nol) |
| Margin jasa | 60–80% (waktu kerja) |
| Margin MoR-aaS | net fee 5–10% gross |

**Implikasi:** karena COGS ~0, **setiap rupiah AOV tambahan ≈ profit**. Fokus =
naikkan **AOV (bundle)** + **frekuensi (subscription)** + **rev-share (reseller/MoR)**.
