## 1. Strategi GTM (ringkas)

**Product-led + content-led, Indonesia-first.** Tunjukkan **hasil nyata** (app live dari
1 perintah), bukan janji. Tangkap lead via waitlist, konversi via tripwire, naikkan LTV
via bundle → continuity → high-ticket.

> Pelajaran CLAFTS (Anthropic GTM): akurasi + voice tuning + iterasi = tool GTM yang diadopsi.

## 4. Funnel & konversi

```
Awareness (konten/SEO) → Interest (landing/katalog)
  → Lead (waitlist POST /api/waitlist)
  → Tripwire (skill 59k) → Core (skill 99k / Bundle 990k)
  → Continuity (Founder Pass 149k/bln) → High-ticket (Done-for-You)
```

| Tahap | Metrik | Target |
|---|---|---|
| Landing → katalog | CTR | > 30% |
| Katalog → product | CTR | > 20% |
| Product → checkout | CTR | > 8% |
| Checkout → paid | conv | > 25% |
| Waitlist → first purchase | conv | > 10% |

## 5. Pesan & copy (Indonesia-first)

**Headline pattern:** *manfaat + kecepatan + lokal*
- "Dari ide ke aplikasi live — satu perintah. Bayar pakai Duitku."
- "36 skill agentik siap-pakai. Bahasa Indonesia. Harga IDR."
- "Bukan sekadar prompt — sistem agentik yang langsung jalan."

**Trust strip (wajib di checkout):** logo QRIS/VA/e-wallet + disclosure MoR Oasis BI Pro.

**Objection handling:**
| Keberatan | Jawaban |
|---|---|
| "Cuma prompt ya?" | "Bukan — sistem lengkap boot→deploy→verify." |
| "Bayarnya ribet?" | "QRIS/VA/e-wallet via Duitku, IDR." |
| "Aman & legal?" | "Merchant-of-Record Oasis BI Pro, PJP diawasi BI." |
