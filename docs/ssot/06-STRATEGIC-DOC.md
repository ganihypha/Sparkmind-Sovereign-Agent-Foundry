## 1. Posisi pasar (positioning statement)

> Untuk **builder & UMKM Indonesia** yang ingin menjalankan bisnis dengan AI agent,
> **SparkMind-OBP** adalah **marketplace skill agentik siap-pakai berbahasa Indonesia**
> yang — berbeda dari kumpulan prompt atau tool agentik USD/English — menyediakan
> **sistem agentik lengkap (boot→orchestrate→deploy→verify)** dengan **payment lokal
> patuh (Duitku/MoR)**.

**Kategori:** "Sovereign Agent Foundry" (kategori baru — bukan "prompt marketplace",
bukan "SaaS tool", tapi **agentic capability marketplace lokal**).

## 3. Analisis GAP (di mana kompetitor LEMAH)

1. **Gap eksekusi:** kompetitor jual "ide/prompt", bukan **sistem yang menjalankan
   pipeline nol→production** (`fullstack-cycle`). → kita isi.
2. **Gap lokalisasi:** hampir semua English + USD. → kita Indonesia-first + IDR.
3. **Gap payment patuh:** kreator ID sulit menerima uang lintas-brand secara legal. →
   kita punya **MoR (OBP) + Duitku** (diawasi BI).
4. **Gap kurikulum→produk:** course mengajar tapi tak memberi produk; tool memberi
   produk tapi tak mengajar. → kita gabung (skill + Canon docs).
5. **Gap governance:** sedikit yang menawarkan **zero-trust / credit-aware / verify-rubric**
   untuk agent. → kita punya skill governance.
