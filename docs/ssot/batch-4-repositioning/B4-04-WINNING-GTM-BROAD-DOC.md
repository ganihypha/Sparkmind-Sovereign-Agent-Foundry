## 1. Strategi GTM (ringkas)

**Outcome-led + proof-led + partner-led, Indonesia-first.**
- **Outcome-led:** semua materi menunjukkan *hasil* ("salon ini sekarang full-booking"),
  bukan fitur teknis.
- **Proof-led:** case study lokal & demo "app jadi di depan mata" sebagai bukti.
- **Partner-led:** agency/komunitas/asosiasi sebagai **pengganda distribusi** (kita 1-operator).

> Berbeda dari GTM niche (konten teknis utk developer), GTM ini berbicara ke **pemilik usaha**
> lewat **bahasa masalah-mereka** dan **saluran tempat mereka sudah berkumpul**.

---

## 2. Mengapa kita menang (vs lawan nyata)

| Lawan | Cara kita menang |
|---|---|
| **Freelancer/agensi (lambat, mahal)** | hasil dalam hari + harga transparan + langganan support |
| **SaaS asing (mahal, Inggris)** | IDR, Bahasa Indonesia, QRIS/VA, fitur lokal, bisa custom |
| **Marketplace (no brand/data)** | toko & data milik pelanggan sendiri |
| **No-code lokal (rakit sendiri)** | kita kirim hasil-jadi + opsi DIY untuk yang mau |

**Unfair advantages (di pasar luas):**
1. **Delivery diakselerasi agent** → 1 operator setara tim → margin & kecepatan.
2. **MoR + Duitku patuh sudah jalan** → bisa terima uang lokal lintas-brand (langka).
3. **Stack edge ~Rp 0 COGS** → bisa harga agresif & tetap untung.
4. **Dua pintu (mainstream + developer)** → top-of-funnel proof gratis dari komunitas builder.

## 3. Channel (saluran ke pasar massal)

| Channel | Taktik | Untuk ICP |
|---|---|---|
| **Konten "hasil nyata"** | reel/video before-after: "warung ini sekarang punya app" | semua |
| **Komunitas usaha** | grup WA/Telegram UMKM, asosiasi (barbershop, EO, yayasan) | ICP-1/5 |
| **Agency reseller** | rekrut agensi sbg white-label → mereka jual ke klien mereka | ICP-3 (channel) |
| **Marketplace lokal/loka-karya** | bazar UMKM, event Komdigi/SAPA-UMKM, workshop | ICP-1/2 |
| **SEO lokal** | "aplikasi kasir salon", "bikin toko online murah", per-kota | ICP-1/2 |
| **KOL mikro UMKM** | endorse murah, testimoni lokal | semua |
| **Jalur developer (proof)** | konten "build in public", komunitas dev → bukti + supply kreator | ICP-0 |
| **Referral** | pelanggan & partner dapat komisi/kredit | semua |

> **Prioritas akuisisi murah:** komunitas + agency reseller + konten hasil. Hindari iklan
> berbayar mahal di awal (credit-aware).

## 4. Funnel (pasar massal)

```
Awareness  : konten hasil nyata + SEO lokal + komunitas/partner
  → Interest : landing per-vertikal (outcome, bukan jargon)
  → Lead     : waitlist / konsultasi gratis (POST /api/waitlist, /api/intake)
  → Tripwire : eBook/mini-kelas 49k  ATAU  setup vertikal 199k
  → Core     : produk inti + langganan (Care Plan / AI Staff)   ← MRR
  → High-tkt : App Custom / AI Company in a Box
  → Partner  : reseller / MoR-aaS / franchise                   ← network effect
```

| Tahap | Metrik | Target awal |
|---|---|---|
| Landing → lead | conv | > 5% |
| Lead → tripwire/setup | conv | > 10% |
| Setup → langganan aktif (bln-2) | retensi | > 60% |
| Pelanggan → high-ticket/upsell | conv | > 8% |
| Partner aktif | jumlah | ≥ 1 (D90) |

## 5. Copy & pesan (Indonesia-first, non-teknis)

**Headline pattern:** *masalah + hasil + kecepatan + lokal*

- "Bisnis jasa-mu masih atur booking di chat? **Punya app sendiri, jadi dalam sehari.**"
- "**Toko online + CS otomatis.** Bayar pakai QRIS. Bahasa Indonesia. Harga IDR."
- "Nggak ngerti AI? **Tenang — kamu cukup punya masalah, kami yang bangun solusinya.**"
- "Software asing kemahalan? **Versi lokalnya: ratusan ribu, bukan jutaan/bulan.**"

**Trust strip (wajib):** logo QRIS/VA/e-wallet + disclosure MoR (OBP) + kebijakan refund/privasi.

**Voice:** lugas, hangat, membantu, anti-hype (D-1 Truth-Lock). Jargon hanya di halaman developer.

**Tes 10-detik:** jika pemilik warung tak paham hero dalam 10 detik → copy gagal, perbaiki.

## 6. Program partner (mesin distribusi 1-operator)

| Program | Mekanisme | Insentif |
|---|---|---|
| **Agency White-Label** | agensi jual katalog dgn brand sendiri | rev-share 30–50% |
| **MoR-aaS** | kreator/SMB pakai rel bayar kita | fee 5–10% |
| **Franchise vertikal** | operator lokal jalankan sub-brand di kota | lisensi + rev-share |
| **Referral** | siapa pun rujuk pelanggan | komisi/kredit |

> Partner = cara skala distribusi tanpa menambah headcount. Tiap partner menambah jangkauan
> ke pasar luas + memperkaya brand_ledger (memperkuat moat compliance).

## 10. Anti-pola (cara kalah — dihindari)

- ❌ Tetap berbicara "skill/orchestrator" di hero publik → pasar luas tak paham.
- ❌ Bakar uang iklan sebelum punya case study → boros, tak credit-aware.
- ❌ Janji hasil yang tak bisa di-deliver → langgar Truth-Lock → rusak trust.
- ❌ Buang jalur developer → kehilangan proof gratis & supply kreator.
- ❌ Harga seperti komoditas (jual file murah) → kembali ke jebakan niche.

> Lanjut: pemetaan teknis aset lama → produk baru & urutan eksekusi → **B4-05 MIGRATION-MAP**.
